# ADR-002

## Title

Frontend Technology Stack

---

## Status

Accepted

---

## Decision

The ADQ frontend will use:

- React
- TypeScript
- Vite
- Tailwind CSS
- ESLint
- Prettier

---

## Reason

- Maintainability
- Scalability
- Modern Tooling
- Excellent Community Support
- Type Safety
- Component Reusability
